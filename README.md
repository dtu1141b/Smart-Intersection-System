# Smart Intersection System

## Overview
This project was developed as part of the course **CS F214: Logic in Computer Science** during undergraduate studies at **BITS Pilani, Pilani Campus**, and was awarded **1st Rank**.

## Description
The Smart Intersection System focuses on applying logical reasoning and formal methods to model and analyze traffic intersection behavior, ensuring correctness, safety, and efficiency in decision-making.

## Implementation Details
For a detailed explanation of the system design, logical formulation, algorithms used, and implementation specifics, please refer to the document:

**`SmartIntersectionSystem.pdf`**

## Technologies & Concepts
- Propositional and First-Order Logic  
- Formal Modeling  
- Logical Constraints and Verification of LTL & CTL properties
